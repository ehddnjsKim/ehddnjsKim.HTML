    timetable.html 이동하는 하이퍼링크 만들기

timetable.html: 시간표 페이지
- <table>을 사용하여 시간표 작성(주간 시간표(개인))
- 요일별 구조
- colspan, rowspan 중 하나 이상 필수 사용
- <thead>, <tbody> 구조 구분
CSS 사용하여 테두리, 정렬, 배경색, 셀 간격 등 꾸미기
- <caption> 태그가 무엇인지 알아보고 적용하기

- 지금까지 배운  HTML, CSS의 기초 문법 및 레이아웃을 실습을 통해 복습

- 보유 기술 스택
- 추가로 하고 싶은 말

- CSS Grid 또는 Flex를 사용하여 레이아웃을 구성
- 외부 CSS
